About Loan Predictor
Built to Make Decisions Transparent

Loan Predictor is an academic machine learning project developed by Team Zyphra. It demonstrates how AI can assist in financial decision-making by predicting whether a loan application is likely to be approved or rejected.

The model was trained on a dataset of 45,000+ loan records containing applicant demographics, income details, credit history, and loan parameters. We used a Decision Tree Classifier — chosen for its interpretability and strong performance on tabular financial data.

The project covers data cleaning, exploratory analysis, feature engineering, model training and evaluation, and deployment as a live web application using Flask.

Python pandas NumPy scikit-learn Flask Decision Tree HTML
